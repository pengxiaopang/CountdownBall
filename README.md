# 🕒 Countdown Ball - 悬浮倒计时球

> 一个轻量级、高颜值的桌面悬浮倒计时工具，支持多种模式切换，助你高效工作、专注演示、轻松“牛马”。


## 🌟 功能亮点

- ✅ **常规模式**：自由设置倒计时，支持暂停/继续/重置
- ✅ **幻灯片模式（PPT 演示）**：自动计时，专注演讲不看表
- ✅ **牛马模式**：午休 / 下班倒计时，双击切换显示「实时牛马费」（趣味彩蛋）
- ✅ **智能提醒系统**
  - 时间归零时闪烁 + 弹窗提醒
  - 临界值颜色渐变提示（红/黄预警）
- ✅ **系统常驻功能**
  - 开机自启
  - 隐藏到托盘
  - 右键菜单快捷操作
- ✅ **快速入口管理**
  - 添加常用文件夹/程序路径
  - 快速打开，提升效率


## 🛠️ 使用方式

1. 下载并运行程序
2. 双击球体进入不同模式
3. 右键点击弹出菜单：
   - 切换模式
   - 设置主题颜色、球体大小
   - 管理快捷入口
   - 开机自启 / 退出程序

## ⚙️ 配置说明

| 功能 | 说明 |
|------|------|
| **球体大小** | 支持滑块调节，动态缩放 |
| **主题颜色** | 自定义背景色与进度条颜色 |
| **快捷入口** | 添加常用路径，一键访问 |
| **开机自启** | 启用后随系统启动 |

## 📦 技术栈

- Python 3.8+
- PyQt5（GUI）
- Qt Widgets & Graphics View
- 配置持久化（env）
- 系统托盘 & 自启动支持

## 🧩 特色彩蛋

👉 **牛马模式下的“实时牛马费”**：
